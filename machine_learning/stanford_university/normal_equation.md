mathmatic equation:
    X * theta = Y => transpose(X) * X * theta = transpose(X) * Y => theta = inverse(transpose(X) * X) * transpose(X) * Y
    reference to https://www.coursera.org/learn/machine-learning/supplement/bjjZW/normal-equation

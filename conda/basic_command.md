# [basic command]()
```sh
# create an env
conda create --name planercnn
# enable an env
conda activate planercnn
# disable en env
conda deactivate

# update conda
 conda update -n base -c defaults conda

```
# [command in env]()
```sh
# install a specified version of python
conda install -y -c anaconda python=3.7
conda install -y pytorch=0.4.1
```
